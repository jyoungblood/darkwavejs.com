Install the [vite ngrok plugin](https://github.com/aphex/vite-plugin-ngrok)
- `npm i -D vite-plugin-ngrok`


Add to astro.gonfig (in `vite.plugins[]`)
  - `import { ngrok } from "vite-plugin-ngrok";`
  - `
      ngrok({
        domain: import.meta.env.VITE_NGROK_DOMAIN,
        compression: true,
        authtoken: import.meta.env.VITE_NGROK_AUTH_TOKEN,
      }),
  `


Add to .env
  - `
    VITE_NGROK_DOMAIN="XXXXXX"
    VITE_NGROK_AUTH_TOKEN="XXXXXX"  
  `
