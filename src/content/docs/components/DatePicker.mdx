Date and time selection component using [Flatpickr](https://flatpickr.js.org/).

## Basic Usage

```astro
<DatePicker
  label="Date"
  name="date"
  dateFormat="Y-m-d"
  altInput={true}
  altFormat="F j, Y"
/>
```

## Date and Time

```astro
<DatePicker
  label="Date and Time"
  name="datetime"
  enableTime={true}
  dateFormat="Y-m-d H:i"
  altInput={true}
  altFormat="F j, Y h:i K"
/>
```

## Date Range

```astro
<DatePicker
  label="Date Range"
  name="daterange"
  mode="range"
  dateFormat="Y-m-d"
  altInput={true}
  altFormat="F j, Y"
/>
```

## With Constraints

```astro
<DatePicker
  label="Select Date"
  name="date"
  dateFormat="Y-m-d"
  minDate="2024-01-01"
  maxDate="2024-12-31"
/>
```

## Props

| Prop | Type | Description |
|------|------|-------------|
| `label` | string | Input label text |
| `name` | string | Input name attribute |
| `dateFormat` | string | Date format for value |
| `altInput` | boolean | Show friendly display format |
| `altFormat` | string | Display format |
| `enableTime` | boolean | Enable time selection |
| `mode` | string | `single`, `range`, or `multiple` |
| `minDate` | string | Minimum selectable date |
| `maxDate` | string | Maximum selectable date |
| `inline` | boolean | Show inline calendar |
| `weekNumbers` | boolean | Show week numbers |
